# RFT Reproduction: Cross-Entropy Baseline for FLYP on iWildCam-WILDS

Reproducing the cross-entropy baseline from Goyal et al. (2022),
*Finetune Like You Pretrain: Improved Fine-tuning of Zero-Shot Vision Models*.

## Status

🚧 **Work in progress** — Day 1 of project setup (2026-05-21).

## Goal

- Reproduce the CE-loss baseline from FLYP Table 1 (CLIP ViT-B/16 on iWildCam)
- Target metrics: OOD Accuracy ~48%, OOD Macro-F1 ~21%
- Tolerance for "successfully reproduced": ±1 percentage point

## Methodology

Following the [DPS repository](https://github.com/changliu816/DPS) as the reference implementation.
